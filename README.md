# Vue_Blog

> 用Vuepress搭建的一个个人博客，通过github免费托管。

## 起步

安装vuepress yarn
```
npm install -g vuepress yarn 
``` 

clone 官方Demo

```
git clone git@github.com:docschina/vuepress.git
```

启动服务
```
cd vuepress    //进入项目根目录
yarn           //安装依赖
yarn dev       //启动服务
```
